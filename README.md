
TechCorp Simulator is a turn-based business management game built with Java and Spring Boot.

Players manage a software development company by accepting projects, hiring employees, investing in automation, and competing against an AI-controlled rival company.

## Features

* Turn-based business simulation
* AI opponent with Easy, Normal, and Hard difficulty
* Project management (accept, hold, resume, cancel)
* Employees, Freelancer Bots, and Automated Tools
* Dynamic productivity allocation
* Bankruptcy and win/lose conditions
* Available in both Console and Web versions

## How to Play

1. Select an AI difficulty.
2. Accept software development projects.
3. Increase productivity through hiring and automation.
4. Complete projects to earn rewards.
5. Manage cash flow and salary expenses.
6. Beat the AI company or finish with the highest cash balance.

## Run

### Console Version

```bash
mvn compile
```

```bash
mvn exec:java
```

### Web Version

```bash
mvn spring-boot:run
```

Then open:

```text
http://localhost:8080
```
